# 💎 SERA MC - Official Landing Page & Website

Website resmi dan landing page interaktif untuk **SERA MC** Minecraft Server.

## 🚀 Fitur Utama
- **Modern Glassmorphism UI**: Tampilan visual dark mode modern dengan efek ambient glow, crystal sparkles, dan backdrop blur.
- **3D Coverflow Sliders**: Gallery interaktif 3D untuk Ranks Shop & Custom Races.
- **Copy IP Dialog**: Pop-up interaktif untuk menyalin IP Bedrock & Java Edition secara praktis.
- **FAQ Accordion**: Pertanyaan umum pemain dengan animasi accordion smooth.
- **WhatsApp Integration**: Integrasi tombol pembelian rank langsung terhubung ke WhatsApp Admin.
- **Fully Responsive**: Dioptimalkan untuk perangkat Desktop, Tablet, dan Mobile.

## 📁 Struktur Direktori
```text
├── index.html       # Halaman utama HTML
├── style.css        # Custom CSS, animasi, & breakpoint responsive
├── script.js        # Logic interaktif Vanilla JS (Modal, Slider, Toast, FAQ)
├── vercel.json      # Konfigurasi deployment Vercel
├── .gitignore       # Berkas yang diabaikan oleh Git
└── *.png / *.webp   # Aset gambar & logo server
```

## 🌐 Cara Deploy ke GitHub & Vercel

### 1. Push ke GitHub
1. Buka Git Bash / Terminal atau **GitHub Desktop**.
2. Inisialisasi repository dan buat commit:
   ```bash
   git init
   git add .
   git commit -m "Initial commit - SERA MC Landing Page"
   ```
3. Hubungkan ke repository GitHub kamu dan push:
   ```bash
   git remote add origin https://github.com/USERNAME/sera-mc.git
   git branch -M main
   git push -u origin main
   ```

### 2. Deploy ke Vercel
1. Buka [Vercel Dashboard](https://vercel.com/dashboard).
2. Klik **Add New...** -> **Project**.
3. Hubungkan akun GitHub kamu dan pilih repository `sera-mc`.
4. Pilih **Framework Preset**: *Other / HTML, CSS, JS*.
5. Klik **Deploy**. Website kamu akan langsung online dalam beberapa detik!
